# Pictionary

This is a minimal implementation of pictionary game that can be played online with friends.

## Built With
- [React.js](https://reactjs.org/?ref=https://github.com/akshitkrnagpal/pictionary)
- [Vite](https://vitejs.dev/?ref=https://github.com/akshitkrnagpal/pictionary)
- [Tailwind CSS](https://tailwindcss.com/?ref=https://github.com/akshitkrnagpal/pictionary)
- [Socket.io](https://socket.io/?ref=https://github.com/akshitkrnagpal/pictionary)
- [shadcn/ui](https://ui.shadcn.com/?ref=https://github.com/akshitkrnagpal/pictionary)
